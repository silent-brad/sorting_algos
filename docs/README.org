* Testing

Run all tests with:

#+BEGIN_SRC sh
dune test
#+END_SRC

The test suite in =test/test_sorting_algos.ml= verifies each algorithm
against multiple cases: empty arrays, single elements, already sorted,
reverse sorted, duplicates, and random input.
